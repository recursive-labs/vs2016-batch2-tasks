(say pramode_ce, which is my twitter name) and display the last 20 tweets of the

"requests" module in Python http://docs.python-requests.org/en/master/) and the
twitter REST API.

d) Write a very simple django app which will have the provision to enter the
name of a twitter user - the app will then display the last 20 tweets on the
web page. You can use the "jquery" javascript library to display the tweets on
the same page. You should use "twitter bootstrap" (http://getbootstrap.com/2.3.2/) css
framework to style your web page.
